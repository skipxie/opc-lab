# OPC/OPC UA 主题网站 - 页面设计说明（Desktop-first）

## 全局设计规范（适用于所有页面）

### Layout
- 桌面端优先：内容最大宽度 1200px（`max-w-[1200px]`），左右留白自适应。
- 栅格与间距：以 12 栅格思路布局（实现可用 Flexbox + CSS Grid 混合）；区块纵向间距 40–64px。
- 响应式断点：
  - ≥1280px：完整导航 + 双栏内容（正文/目录）
  - 768–1279px：导航折叠为更紧凑布局，正文优先
  - <768px：导航抽屉/汉堡菜单，目录折叠为可展开面板

### Meta Information（通用模板）
- Title：`{页面名} | OPC/OPC UA 主题网站`
- Description：一句话概括该页内容（80–120 字）。
- Open Graph：
  - `og:title`、`og:description` 与 Title/Description 对齐
  - `og:type=website`
  - `og:image=/og-cover.png`（统一封面）

### Global Styles（Design Tokens）
- 背景色：`#0B1220`（深色主背景）/ 内容卡片 `#111B2E`
- 主题色（Primary）：`#2F81F7`
- 强调色（Accent）：`#22C55E`（成功/推荐）、`#F59E0B`（提醒）
- 文本色：主文本 `#E5E7EB`，次级 `#9CA3AF`
- 字体：
  - 标题：`font-semibold`，H1 40px、H2 28px、H3 20px
  - 正文：16px，行高 1.7
  - 代码：等宽字体（用于协议字段/伪代码/配置示例）
- 按钮：
  - Primary：主题色底 + 白字；hover 加深 8% + 轻微阴影
  - Secondary：描边按钮；hover 背景浅填充
- 链接：默认主题色；hover 下划线 + 亮度提升
- 卡片：圆角 12px；边框 1px 半透明；hover 上浮 2px

### 通用组件
- 顶部导航（Navbar）：Logo + 主导航（介绍/场景/资源/联系）+ 右侧“资源中心”按钮
- 页脚（Footer）：版权、免责声明、隐私说明（简版）、外链（可选）
- 内容目录（TOC）：右侧吸附（桌面端）；移动端折叠
- 代码/提示块：
  - Note（蓝色）、Warning（橙色）、Tip（绿色）

## 页面 1：首页（/）

### Layout
- 纵向分区堆叠（Stacked Sections）；关键区块内使用 2–3 列卡片 Grid。

### Meta
- Title：`首页 | OPC/OPC UA 主题网站`
- Description：介绍网站定位、提供入门导航与推荐资源。

### Page Structure
1. Navbar
2. Hero 区
3. 快速入口（卡片）
4. 推荐内容/最新资源
5. 价值背书与适用人群
6. Footer

### Sections & Components
- Hero
  - 左侧：H1（站点定位）、简短说明（互操作/安全/信息模型/落地）、两个 CTA（进入介绍 / 资源中心）
  - 右侧：概念级示意图（可用静态插画/简化拓扑）
- 快速入口卡片（4–6 张）
  - 组件：Icon + 标题 + 1 行说明
  - 交互：hover 高亮；点击跳转到对应页面并带锚点（如 /intro#security）
- 推荐内容/最新资源
  - 组件：列表（标题、摘要、更新时间、标签）
  - 交互：点击进入资源详情或跳到介绍页相关段落
- 价值背书
  - 组件：三列要点（“为什么选 OPC UA”“适用场景”“常见误区”）

## 页面 2：OPC/OPC UA 介绍（/intro）

### Layout
- 桌面端：双栏布局（左：正文 8/12；右：目录/快捷跳转 4/12）。
- 正文采用“章节化文档”样式，支持锚点导航。

### Meta
- Title：`OPC/OPC UA 介绍 | OPC/OPC UA 主题网站`
- Description：解释 OPC/OPC UA 核心概念、通信模型、安全与信息建模。

### Page Structure
1. 页面标题区（标题 + 简介 + 章节标签）
2. 文档正文（分章节）
3. 页内导航（TOC）
4. 相关资源推荐（底部）

### Sections & Components
- 章节 1：概念与术语
  - 组件：术语表（Term + Explanation）；支持搜索过滤（本页内）
- 章节 2：架构与通信模型
  - 组件：对比卡片（Client/Server vs PubSub）
  - 组件：流程图/时序示意（静态图或 Mermaid 渲染结果截图）
- 章节 3：信息模型与地址空间
  - 组件：树形结构示意（概念级 Node 层级）
  - 组件：代码块（示例字段/伪代码，仅说明用途）
- 章节 4：安全机制概览
  - 组件：清单（证书、信任、加密/签名、用户认证方式）
  - 组件：Warning 提示（常见配置误区）
- 底部推荐
  - 组件：资源卡片（指向 /resources 并带筛选参数）

## 页面 3：应用场景与解决方案（/solutions）

### Layout
- 顶部筛选 + 下方卡片网格；点击后进入同页抽屉/详情区（减少页面数量）。

### Meta
- Title：`应用场景与解决方案 | OPC/OPC UA 主题网站`
- Description：按目标与行业组织的落地方案、参考架构与排障建议。

### Page Structure
1. 页面标题与筛选栏
2. 场景卡片列表（Grid）
3. 场景详情（同页展开：Drawer/Accordion）
4. FAQ 与排障（按症状分组）

### Sections & Components
- 筛选栏
  - 控件：关键词输入、行业下拉、目标标签（采集/上云/网关/集成）
- 场景卡片
  - 字段：场景名、适用对象、推荐通信模型、复杂度标签
  - 交互：点击展开详情（不离开页面）
- 场景详情模板
  - 区块：背景/目标、推荐拓扑（示意图）、数据建模建议、安全建议、实施清单（Checklist）
- FAQ 与排障
  - 组件：Accordion（症状 → 可能原因 → 处理步骤）

## 页面 4：资源中心（下载与链接）（/resources）

### Layout
- 桌面端：左侧筛选侧栏 + 右侧资源列表（Sidebar + List）。

### Meta
- Title：`资源中心 | OPC/OPC UA 主题网站`
- Description：提供资料、工具、示例与外部链接，支持搜索与筛选下载。

### Page Structure
1. 顶部搜索与排序
2. 左侧筛选（类型/标签/更新时间）
3. 资源列表（卡片或表格视图切换）
4. 资源详情（Modal 或右侧抽屉）

### Sections & Components
- 搜索与排序
  - 控件：搜索框、排序（更新时间/相关性）
- 筛选侧栏
  - 控件：类型（doc/tool/sample/link）、标签多选、更新时间范围
- 资源列表
  - 卡片字段：标题、摘要、标签、更新时间、来源（外链标识）
  - 交互：点击打开详情；支持复制链接
- 资源详情（Modal/Drawer）
  - 展示：简介、适用对象、版本、checksum（可选）、来源链接
  - CTA：下载文件（若 `file_path` 存在）或“前往来源”

## 页面 5：联系与合作（/contact）

### Layout
- 双栏布局：左侧表单，右侧联系信息与说明卡片。

### Meta
- Title：`联系与合作 | OPC/OPC UA 主题网站`
- Description：提交咨询需求，了解合作方式与站点声明。

### Page Structure
1. 标题与说明
2. 咨询表单
3. 联系信息卡片
4. 免责声明/隐私说明（简版）

### Sections & Components
- 咨询表单
  - 字段：姓名、公司（可选）、邮箱、主题、内容
  - 交互状态：
    - 校验：邮箱格式、内容最小长度
    - 提交中：按钮 loading
    - 成功：提示“已收到，预计 X 个工作日回复”
    - 失败：提示可重试，并给出备用邮箱
  - 反垃圾：隐藏蜜罐字段 + 前端节流（同设备短时间内限制提交）
- 联系信息
  - 展示：邮箱、工作时间、合作方向（如培训/咨询/方案评估）
- 声明
  - 文案：资源来源说明（外链为主）、不保证第三方内容可用性、隐私简述
